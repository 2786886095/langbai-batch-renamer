# 浪白重命名工具 v1.1.4

本版本修复载入已保存命名方案后，命名格式错误显示为 `BatchRename.Core.RenamePreset` 的问题。

## 修复与改进

- 为命名方案增加明确的模板显示文本，选中方案后输入框显示实际命名格式，例如 `{1}{S}`。
- 下拉列表继续显示“方案名称 + 模板”，不影响直接编辑命名格式和删除方案。
- 显示文本不写入方案 JSON，现有本地方案无需迁移即可继续使用。
- 扩展 UI 自动化脚本，可展开方案列表、选择方案并验证编辑框没有显示内部类型名。

## 验证

- Release 构建：0 个警告、0 个错误。
- 自动化测试：20/20 通过。
- WPF 回归测试：方案写入 JSON、重新读取并选入可编辑下拉框后，显示 `{1}{S}`。
- 真实窗口验证：选择本地方案 `1-n` 后，命名格式显示 `{1}{S}`，状态提示正确显示已载入方案。

## 安装

下载 `BatchRename-Setup-1.1.4-x64.exe`，右键选择“以管理员身份运行”。安装程序会注册资源管理器右键菜单并重启一次资源管理器。

安装包使用项目本地自签名证书，Windows 可能显示“未知发布者”或 SmartScreen 提示。请从本仓库 Release 页面下载，并核对下面的 SHA-256。
