# 浪白重命名工具 v1.1.5

本版本修复软件内更新后已保存的命名方案、设置和回退历史消失的问题。

## 修复与改进

- 更新安装改用 MSIX 原位升级，不再先执行 `Remove-AppxPackage` 卸载旧版本。
- 原位升级会保留应用包的本地数据，包括命名方案、更新设置和重命名历史。
- 同版本重新安装仍可正常完成，不需要删除旧包数据。
- 新增安装脚本契约测试，禁止更新流程重新引入先卸载再安装的行为。
- 新增真实更新保留测试脚本，可验证升级、恢复和测试标记清理结果。

## 验证

- Release 构建：0 个警告、0 个错误。
- 自动化测试：21/21 通过。
- 真实 Windows MSIX 更新：v1.1.4 → v1.1.5，本地包数据完整保留。
- 真实恢复验证：v1.1.5 → v1.1.4，本地包数据仍然保留，测试标记已删除。
- 同版本 v1.1.4 重新安装验证通过。

以前更新时已经被系统删除的方案无法由本版本自动恢复；从 v1.1.5 开始的软件内更新将保留现有数据。

## 安装

下载 `BatchRename-Setup-1.1.5-x64.exe`，右键选择“以管理员身份运行”。安装程序会注册资源管理器右键菜单并重启一次资源管理器。

安装包使用项目本地自签名证书，Windows 可能显示“未知发布者”或 SmartScreen 提示。请从本仓库 Release 页面下载，并核对下面的 SHA-256。
