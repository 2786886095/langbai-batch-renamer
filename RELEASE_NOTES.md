# 浪白重命名工具 v1.0.0

首个公开版本，可从 Windows 文件资源管理器右键菜单直接批量重命名文件和文件夹。

## 功能

- 文件与文件夹混合批量重命名，并按名称自然排序。
- 支持 `{P}`、`{S}`、`{T}`、`{N}`、`{zN}`、`{0}`、`{1}`、`{z8}` 模板。
- 支持普通查找替换和正则表达式。
- 执行前预览、逐项编辑、名称冲突检查。
- 保留最近 50 次操作，可在重启后回退。
- 支持 Windows 10 1809 及更高版本、Windows 11（x64）。

## 安装

下载 `BatchRename-Setup-1.0.0-x64.exe`，右键选择“以管理员身份运行”。安装程序会注册资源管理器右键菜单并重启一次资源管理器。

安装包使用项目本地自签名证书，Windows 可能显示“未知发布者”或 SmartScreen 提示。请从本仓库 Release 页面下载，并核对下面的 SHA-256。
