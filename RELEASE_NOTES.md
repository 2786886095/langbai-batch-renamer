# 浪白重命名工具 v1.1.3

本版本修复保存命名方案时输入框被压缩、文字不可见的问题。

## 修复与改进

- 保存命名方案弹窗改为根据内容自动调整高度，说明文字换行时不再挤压输入区域。
- 方案名称输入框固定为 38 像素高度，文字、选中状态和圆角边框均可完整显示。
- 空错误提示不再占用布局空间；未填写名称时仍会正常显示中文校验信息。
- 为保存按钮和方案名称输入框补充自动化标识，并增加运行时布局检查，输入框低于 34 像素时验证会直接失败。

## 验证

- Release 构建：0 个警告、0 个错误。
- 自动化测试：19/19 通过。
- 真实 WPF 窗口截图验证：方案名称输入框高度 38 像素，预置名称可见，边框完整。

## 安装

下载 `BatchRename-Setup-1.1.3-x64.exe`，右键选择“以管理员身份运行”。安装程序会注册资源管理器右键菜单并重启一次资源管理器。

安装包使用项目本地自签名证书，Windows 可能显示“未知发布者”或 SmartScreen 提示。请从本仓库 Release 页面下载，并核对下面的 SHA-256。
