# 浪白重命名工具 v1.1.1

本版本重点强化回退事务、历史数据保护和复杂正则表达式的安全边界。

## 新增与改进

- 回退操作现在与历史状态写入组成补偿事务：若历史状态无法保存，会自动恢复到回退前的名称，避免磁盘状态和历史记录不一致。
- 损坏的历史文件不会再被新增记录静默覆盖；写入会安全停止并触发重命名回滚，保留原始文件供检查或恢复。
- 正则查找替换增加执行超时保护，复杂表达式不会再让预览界面长时间无响应。
- 新增 10,000 项预览、超过 260 字符总路径、UNC 网络共享执行与回退，以及回退历史故障注入测试。
- 命名方案可保存到本机，支持载入、同名覆盖和删除；规则、替换、序号、时间格式与排序设置会一起保存。
- 新增按名称、修改日期、文件大小、文件类型排序，并支持倒序。
- 新增独立“最终重命名预览”窗口；只有在窗口中再次确认后才会改动磁盘。
- 原名称使用灰色只读区域，新名称使用蓝色可编辑区域，并用方向箭头明确区分。
- 时间格式、起始序号和补零位数加入用途与动态示例说明。
- 修复鼠标停在下拉框上滚动时意外切换选项的问题；滚轮现在只滚动规则面板。
- 编辑框加入中文右键菜单：撤销、剪切、复制、粘贴、删除、全选。
- 超过 Windows 单项名称上限的结果会在预览阶段直接阻止，不再等到执行时失败。
- 历史文件被占用、损坏或暂时无权限读取时，历史窗口会安全降级为空列表。
- 重命名完成但回退记录无法保存时，会自动恢复原名称；若恢复也失败，会明确提示立即检查磁盘状态。
- 修复 Windows PowerShell 5.1 构建中文 MSIX 清单时的 UTF-8 编码问题。

## 安装

下载 `BatchRename-Setup-1.1.1-x64.exe`，右键选择“以管理员身份运行”。安装程序会注册资源管理器右键菜单并重启一次资源管理器。

安装包使用项目本地自签名证书，Windows 可能显示“未知发布者”或 SmartScreen 提示。请从本仓库 Release 页面下载，并核对下面的 SHA-256。
